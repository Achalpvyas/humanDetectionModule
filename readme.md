## Overview

Human Detection module for ACME Robotics with:

- cmake
- googletest

## Standard install via command-line
```shell script
git clone --recursive https://travis-ci.org/urastogi885/Human-Detection
cd <path to repository>
mkdir build
cd build
cmake ..
make
./test/cpp-test
./app/shell-app
```
